    - 유산소 운동 (HR Max, HRrest, HRreserve) 최대심박수, 안정심박수, 여유심박수
        - 예: 25세 안정시 심박수 70회/분
            - 70~85% 최대 심박수 : (220-25)*(0.7~ 0.85)
            - 70~85% 여유 심박수 : (최대심박수-안정시 심박수)*(0.7~ 0.85)
            - 70~85%강도의 목표 심박수 : (최대심박수-안정시 심박수)*(0.7~0.85) + 안정시 심박수
    - 저항 운동 : 횟수 ( RM ) 
    - 유연성 운동 - 정적 동작 : 초, 동적 동작 : 횟수
